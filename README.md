## Chronic Kidney Disease (CKD) Prediction System
## 📌 Project Overview
This project focuses on predicting Chronic Kidney Disease (CKD) using data analytics and basic machine learning techniques. The goal is to analyze patient data and identify whether a person has CKD or not.

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Analyze medical data using EDA
- Build a prediction model
- Visualize results for better understanding

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📊 Dataset
- The dataset contains medical attributes such as age, blood pressure, sugar levels, etc.
- Data was cleaned and processed before analysis.

## ⚙️ Process
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Selection  
4. Model Building using Voting Classifier  
5. Result Visualization  

## 📈 Results
- Built a classification model to predict CKD
- Visualized prediction outcomes (CKD vs Non-CKD)
- Compared actual vs predicted results using graphs

- Improve model accuracy
